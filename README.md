#Texas Hold'em Project

This project is a command-line Texas Hold'em game written in Python. It uses separate files for cards, decks, players, hand evaluation, and the game itself.

## Files
- `card.py`: creates card objects
- `deck.py`: creates and shuffles a deck
- `player.py`: stores player information
- `hand_evaluator.py`: checks poker hands
- `game.py`: runs the game
- `tests/`: contains test files

## How to run the game
```bash
python game.py Alice Bob
```

## How to run tests
```bash
pytest
```
